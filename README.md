# test_langchani
# LangChain Experiments & Chatbot System 🚀

This repository contains experiments and implementations using **LangChain**, including chat models, embeddings, JSON output parsing, and LLM-based chatbot workflows.

The goal of this project is to explore how modern **LLM pipelines** can be structured using modular components such as:

* Chat Models
* Embedding Models
* Output Parsers
* Structured JSON responses
* Modular message handling

---

# 📂 Project Structure

```
test_langchani/
│
├── ChatModels/          # Chat model integrations
├── EmbeddedModels/      # Embedding model implementations
├── LLMs/                # LLM experiment scripts
│
├── app.py               # Main application entry
├── chatbot.py           # Chatbot logic
├── messages.py          # Message schema / message classes
│
├── json_schema.json     # JSON structure definition
├── jsonoutputparser.py  # Structured output parsing
│
├── .gitignore
├── README.md
```

# ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Tilakvasani/test_langchani.git
```

Move into project directory:

```
cd test_langchani
```

Create virtual environment:

```
python -m venv myenv
```

Activate environment:

Windows

```
myenv\Scripts\activate
```

Install dependencies:

```
pip install -r requirements.txt
```

---

# ▶️ Running the Project

Run the main application:

```
python app.py
```

Or run chatbot module:

```
python chatbot.py
```
